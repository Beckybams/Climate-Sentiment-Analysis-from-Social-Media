Climate-Sentiment-Analysis-from-Social-Media
Project Overview

Climate-Sentiment-Analysis-from-Social-Media is an AI-powered system that analyzes public sentiment on climate-related topics using social media data. The project uses Natural Language Processing (NLP) to classify posts into positive, negative, or neutral categories, helping researchers, policymakers, and organizations understand how people perceive climate change issues.

Key Features

Collects and processes social media climate-related text.

Performs sentiment analysis using machine learning or deep learning models.

Generates sentiment scores and labels.

Supports visualization of public opinion trends.

Allows export of results to Excel for reporting.

Technologies Used

Python

Pandas

Scikit-learn / NLP models

NLTK / SpaCy

OpenPyXL (for Excel export)

How It Works

Collects text posts from social media (or uses synthetic data).

Cleans and preprocesses the text.

Applies sentiment analysis to classify public opinion.

Stores results for visualization or export.
